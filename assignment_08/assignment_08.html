tanlily@usc.edu

Topic

This is a random tarot generator project. Users can select the desired topic(love life, career, and personal life) and generate an array of 3-4 tarot cards based on selected topic.In each tarot generation, users can click on each card in the array for more detailed explanation. There will also be a separate page for array explanation for each topic.

Target Audience

People who has question on their love life, career, personal life, or simply interested in tarot card. Major age target range from 16-35.

Design & Style

The major design style will be tarot card in the middle, with text below. However, compare to black and dark scene, there will be fun element arong the paged and the color scene is lighter.

Color Palette

Primary color: medium purple

Secondary color: sapphire

Website Inspiration

  • Tarot Bot
  • Zadianz Tarot
  • Tarot Generator

Scope and Section

There will be majorly five pages:

  • Page 1: The first page is the tarot selection page where the users can select the topic they want to know about and their divintion history
  • Page 2-4: Page 2-4 are the shuffling and viewing the card based on selected topics. The user can also view the detailed explanation of each card by click on individual cards
  • Page 5: Page 5 is the page where the users can see the exanation of the array of card from different topics

Extras

  • DOM Manipulation to select topic, shuffle the card, and view the card
  • DOM Storage to save history questions
  • React library to change the card
